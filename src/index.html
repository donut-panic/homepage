INTRO

I promise not to be boring

Hi! My name is Jarek Karczewski and I'm a 30-year-old from Poznań who creates front-end for websites and internet apps. Like many other people in this industry, I also possess other professional experience. However, since the beginning of my professional career I have been associated with the creative industry, and front-end development is for me a natural step on the path of my professional career.

Why front-end development? Because it is an extremely interesting and dynamically developing industry, and I love to learn new things. And I guess that's really the perfect job for me – one that constantly improves my skill and never gets me bored.

I started by learning HTML and CSS (including more advanced issues such as flexbox, CSS grid, and animations). I quickly realized that I need to improve my code writing process. That is why I became interested in SASS and LESS preprocessors and the BEM methodology. In parallel with learning how to create the visual layer, I learned JavaScript (ES6). In it I try to create a code that is optimized and complies with object-oriented programming standards. I also use the React library (add-ons such as Redux), and I plan to learn the Vue.js framework. In addition, I also use GIT.

If you think that my skills will be useful in your team, please contact me.
